# Johnny von Holstein Portfolio

A web-based portfolio for Johnny von Holstein, showcasing GitHub projects across finance tracking, wildlife camera systems, and autonomous lab platforms.

## Projects Featured

- Financial Tracker: React/Vite personal finance dashboard deployed through GitHub Pages.
- Wildlife Camera System: Python dashboard for visit review, camera health, feeder monitoring, and weekly digest workflows.
- Autonomous Labs: Flask and React lab platform for telemetry, automated data upload, anomaly detection, and alerting.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

This repository includes a GitHub Actions workflow that builds the Vite app and publishes it to GitHub Pages.
